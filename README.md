# Django News Website

A simple news website developed using **Python Django**. The project displays news articles, filters them by category, and shows detailed news information.

## Features

* View all news articles
* Category-wise news filtering
* Read detailed news descriptions
* Database integration using SQLite

## Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite

## Run Project

```bash
pip install django
python manage.py migrate
python manage.py runserver
```

Open:
`http://127.0.0.1:8000/`

## Purpose

This project was created to learn Django concepts such as URL routing, views, templates, models, and database operations.
